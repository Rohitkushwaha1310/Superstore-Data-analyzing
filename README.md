📊 End-to-End Data Analytics & Data Science Project
🚀 Overview

This project demonstrates an end-to-end Data Analytics & Data Science workflow using the Superstore dataset. The objective was to transform raw business data into actionable insights through data cleaning, SQL analysis, interactive dashboards, statistical analysis, and machine learning.

🎯 Objectives
Clean and preprocess raw business data.
Perform SQL-based business analysis.
Build an interactive Power BI dashboard.
Validate business decisions using statistical analysis.
Develop machine learning models to predict order profitability.
Segment customers using clustering techniques.
